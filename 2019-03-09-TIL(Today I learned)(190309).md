- Container
	CPU 가상화 기술, 애플리케이션을 구동하기 위한 최소한의 환경을 가상화한다.
    
- Docker
	- 컨테이너 기반의 오픈소스 가상화 플랫폼
    - 도커는 한 애플리케이션을 구동할 수 있는 환경 자체를 가상화(이미지)하여 관리할 수 있다.
- Kubernetes
	컨테이너 관리 서비스
    
- 컨테이너의 장점
	- 하이퍼바이저를 사용하지 않는다.
	- 필요한 만큼씩 자원이 할당되어 효율적인 자원 관리가 가능하다.
	- 서비스를 확장하기 쉽다.
	- 배포하기 쉽다.
